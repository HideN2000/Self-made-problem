問題文
=====

$N$を自然数とする.

「$1 \le i,j \le N」 かつ 「gcd(i,j)\ が \ gcd(N - i,N - j)を割り切る$ 」

を満たす自然数のペア$(i,j)$の個数を求めよ.

なお,$gcd(0,0) = 0$と定義する.


入力
-----
入力は以下の形式で与えられる.
```
N
```

出力
-----
条件を満たすペアの個数を$1$行に出力せよ.

なお,答えは$32bit$整数の範囲に収まるとは限らないことに注意せよ.

最後に改行せよ.

制約
-----

・$1\le N \le 10^6$


サンプル
=====


```入力1
3
```

```出力1
8
```
手計算を行うと,$(i,j) = (2,2)$以外は条件を満たすことがわかる.

したがって,求める個数は$3^2 - 1 = 8$個である.

